# Run Locally

### Setup .env file

```js
DATABASE_URL=...
JWT_SECRET=...
NODE_ENV=...
PORT=...
```

### Install dependencies

```shell
npm install
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
